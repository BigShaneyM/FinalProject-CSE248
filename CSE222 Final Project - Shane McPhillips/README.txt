Data Structures used:

LinkedList(As a queue):
	- Used to store and retrieve college data objects.
	- Big-O values associated with the methods used:
		- Offer - which puts new data at tail end of queue, O(1).
		- Poll - which retrieves and removes data from head of queue, O(1).
		- Peak - which checks the tail end node to see what object is in place, O(1).

ArrayList - used for table view and list view, O(n)
